# 모바일 청첩장

GitHub Pages를 통해 제공되는 모바일 청첩장입니다.

## GitHub Pages 설정 방법

1. GitHub에 새 저장소를 생성합니다
2. 이 코드를 push합니다:
   ```bash
   git add .
   git commit -m "Initial commit: 모바일 청첩장 기초 틀"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git push -u origin main
   ```
3. GitHub 저장소 Settings > Pages로 이동
4. Source를 "Deploy from a branch"로 설정
5. Branch를 "main"으로 선택하고 폴더는 "/ (root)" 선택
6. Save 버튼 클릭

완료되면 `https://YOUR_USERNAME.github.io/YOUR_REPO/` 주소로 접근 가능합니다.

## 로컬 테스트

간단히 `index.html` 파일을 브라우저로 열어서 확인할 수 있습니다.
